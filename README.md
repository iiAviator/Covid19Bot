# Covid19Stats

Discord bot that pulls statistics from the [covid19 api](https://covid19api.com/) and displays them onto Discord. 

## How it works:

|Command|Parameters  | Example |
|--|--|--|
| !world |none | !world|
|!stats|Country Code or Country|!stats United States|
